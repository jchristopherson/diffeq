◆ attempt_step()

procedure, public diffeq::rk_variable_integrator::attempt_step

Attempts a single integration step.

Syntax
subroutine attempt_step( &
class(rk_variable_integrator) this, &
class(ode_container) sys, &
real(real64) h, &
real(real64) x, &
real(real64) y(:), &
real(real64) yn(:), &
real(real64) en(:), &
optional real(real64) xprev(:), &
optional real(real64) yprev(:,:), &
optional real(real64) fprev(:,:), &
optional class(errors) err &
)
Parameters
[in,out]thisThe rk_variable_integrator object.
[in]sysThe ode_container object containing the ODEs to integrate.
[in]hThe current step size.
[in]xThe current value of the independent variable.
[in]yAn N-element array containing the current values of the dependent variables.
[out]ynAn N-element array where the values of the dependent variables at x + h will be written.
[out]enAn N-element array where the values of the error estimates will be written.
[in]xprevAn M-element array containing the previous M values of the independent variable where M is the order of the method. This is typically useful for multi-step methods. In single-step methods this parameter is not used.
[in]yprevAn M-by-NEQN array containing the previous M arrays of dependent variable values where M is the order of the method. This is typically useful for multi-step methods. In single-step methods this parameter is not used.
[in]fprevAn M-by-NEQN array containing the previous M arrays of function (derivative) values where M is the order of the method. This is typically useful for multi-step methods. In single-step methods this parameter is not used.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling.

◆ compute_next_step_size()

procedure, public diffeq::rk_variable_integrator::compute_next_step_size

Computes the next step size.

Syntax
real(real64) function compute_next_step_size( &
class(rk_variable_integrator) this, &
real(real64) hn, &
real(real64) en, &
real(real64) enm1 &
)
Parameters
[in,out]thisThe rk_variable_integrator object.
[in]hnThe current step size.
[in]enThe norm of the error for the current step size.
[in]enm1The norm of the error from the previous step size.
Returns
The new step size.
Remarks
The step size estimate makes use of a PI type controller such that \( h_{n+1} = f h_n \left( \frac{1}{e_n} \right)^{1/k} e_n^{\alpha} e_{n-1}^{\beta}\), where \( f \) is a safety factor, and \( k \) is the order of the integration method.
