◆ compute_jacobian()

procedure, public diffeq::ode_container::compute_jacobian

Computes the Jacobian matrix for the system of ODEs. If a routine is provided with an analytical Jacobian, the supplied routine is utilized; else, the Jacobian is estimated via a forward difference approximation.

Syntax
subroutine compute_jacobian( &
class(ode_container) this, &
real(real64) x, &
real(real64) y(:), &
real(real64) dydx(:), &
real(real64) jac(:,:), &
optional class(errors) err &
)
Parameters
[in,out]thisThe ode_container object.
[in]xThe current independent variable value.
[in]yAn N-element array containing the current dependent variable values.
[in]dydxAn N-element array containing the current derivative values.
[out]jacAn N-by-N matrix where the Jacobian will be written.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • DIFFEQ_MEMORY_ALLOCATION_ERROR: Occurs if there is a memory allocation issue.
  • DIFFEQ_NULL_POINTER_ERROR: Occurs if no ODE function is defined, and the calculation is being performed by finite differences.
  • DIFFEQ_MATRIX_SIZE_ERROR: Occurs if jac is not N-by-N.
